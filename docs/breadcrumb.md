# Breadcrumb 面包屑

显示当前页面的路径，快速返回之前的任意页面。

## 基本用法

适用广泛的基础用法。

<ins-breadcrumb>
<ins-breadcrumb-item to="name">首页</ins-breadcrumb-item>
<ins-breadcrumb-item to="age">组件</ins-breadcrumb-item>
<ins-breadcrumb-item>面包屑</ins-breadcrumb-item>
</ins-breadcrumb>

## 自定义分隔符

<ins-breadcrumb separator="/">
<ins-breadcrumb-item to="name">首页</ins-breadcrumb-item>
<ins-breadcrumb-item to="age">组件</ins-breadcrumb-item>
<ins-breadcrumb-item>面包屑</ins-breadcrumb-item>
</ins-breadcrumb>

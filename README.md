# ins-design

## 使用

### 安装

```bash
yarn install ins-design
```

### 引用

```bash
import Vue from 'vue'
import foxDesign from 'ins-design';
import 'ins-design/lib/ins-design.css';

Vue.use(foxDesign)
```

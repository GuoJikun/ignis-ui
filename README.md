# ins-ui

## 使用

### 安装

```bash
yarn install ins-ui
```

### 引用

```bash
import Vue from 'vue'
import insUI from 'ins-ui';
import 'ins-ui/lib/ins-ui.css';

Vue.use(insUI)
```
